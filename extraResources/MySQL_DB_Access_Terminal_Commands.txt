Access mysql server from terminal:

start the server from folder:

cd /usr/local/mysql-8.0.15-macos10.14-x86_64/support-files

sudo ./mysql.server start

Go to folder where mysql is located:

cd /usr/local/mysql-8.0.15-macos10.14-x86_64/bin

Access server:

./mysql -u root -p

use DLRTM_DB;

show tables;
